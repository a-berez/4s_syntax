# Расширение подсветки синтаксиса 4s для VS Code / Cursor

Это простое локальное расширение, которое добавляет подсветку синтаксиса для файлов формата `.4s`, используемых в [chgksuite](https://chgksuite.pecheny.me/).

Подсвечиваются:

- **идентификаторы полей** (`###LJ`, `###`, `#DATE`, `#EDITOR`, `##`, `#`, `№`, `№№`, `?`, `!`, `=`, `!=`, `/`, `^`, `@`, `-` в начале строки)
- **раздаточные материалы** (в том числе многострочные блоки вида `[Раздаточный материал: ... ]`)
- **картинки** (`(img name.png)`)
- **служебные теги** (`(PAGEBREAK)`, `(screen a|b)`, `4SCOUNTER`)
- **форматирование подчёркиваниями и тильдами**:
  - `_курсив_`
  - `__жирный__`
  - `___подчёркивание___`
  - `____полужирный курсив____`
  - `_____полужирный с подчёркиванием_____`
  - `______полужирный курсив с подчёркиванием______`
  - `~зачёркнуто~` и комбинации
- **продолжения строк**: если поле (например, вопрос или комментарий) продолжается на следующих строках, подсветка сохраняется до следующего поля или пустой строки

Кроме подсветки, расширение добавляет команды форматирования для языка `4s`:

- **4s: Обернуть в _курсив_** (`4s.wrapItalic`) — оборачивает выделение в `_..._` или вставляет пару `_` и ставит курсор внутрь.
- **4s: Обернуть в __жирный__** (`4s.wrapBold`) — оборачивает в `__...__`.
- **4s: Обернуть в ___подчёркивание___** (`4s.wrapUnderline`) — оборачивает в `___...___`.
- **4s: Обернуть в ____полужирный курсив____** (`4s.wrapBoldItalic`) — оборачивает в `____...____`.
- **4s: Обернуть в _____полужирный с подчёркиванием_____** (`4s.wrapBoldUnderline`) — оборачивает в `_____..._____`.
- **4s: Обернуть в ______полужирный курсив с подчёркиванием______** (`4s.wrapBoldItalicUnderline`) — оборачивает в `______...______`.
