## Changelog

### [0.1.1] - 2026-03-04

#### Added
- Горячие клавиши для форматирования в файлах `4s`:
  - `_курсив_`: `Ctrl+Alt+I` / `Cmd+Alt+I`
  - `__жирный__`: `Ctrl+B` / `Cmd+B`, `Ctrl+Alt+B` / `Cmd+Alt+B`
  - `___подчёркивание___`: `Ctrl+U` / `Cmd+U`, `Ctrl+Alt+U` / `Cmd+Alt+U`

### [0.1.0] - 2026-03-04

#### Added

- Первая версия расширения с поддержкой формата 4s (chgksuite):
  - TextMate‑грамматика для полей (`###LJ`, `###`, `#DATE`, `#EDITOR`, `##`, `#`, `№`, `№№`, `?`, `!`, `=`, `!=`, `/`, `^`, `@`, `-`).
  - Подсветка раздаточных материалов (`[Раздаточный материал: ...]`, многострочные блоки).
  - Подсветка картинок (`(img name.png)`), служебных тегов (`(PAGEBREAK)`, `4SCOUNTER`), ссылок и плейсхолдеров.
  - Поддержка продолжений строк для вопросов, ответов, комментариев и источников.
- Сворачивание вопросов по блокам (от `?` до первой пустой строки).
- CodeLens с нумерацией вопросов, включая ручную установку (`№№ N`) и переименование (`№ N`).
- Автодополнение имён файлов картинок в конструкциях `(img ...)` и `(pic: ...)`.
- Базовые команды форматирования:
  - `4s.wrapItalic` (`_курсив_`)
  - `4s.wrapBold` (`__жирный__`)
  - `4s.wrapUnderline` (`___подчёркивание___`)
